# Gerenciardor_Tarefas

API para um Gerenciador de Tarefas.

#Tecnologias

- Node js
- Express
- Mongoose
- Mongo DB
- Nodemon
- Insomnia

#Funções

- Cadastrar Tarefa
- Listar Tarefa
- Colocar Prioridade ( Alta, Média, Baixa )
- Tarefa contém nome, descrição, data de criação, prioridade e concluída ( true or false )
- Listar todas as tarefas
- Marcar tarefas como concluídas
- Listar tarefas com tag de concluídas
- Listar tarefas com tag de pendentes
- Editar Tarefas
- Deletar Tarefas


Futuramente será integrado um front para utilização de usuários.
